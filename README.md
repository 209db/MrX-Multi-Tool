🔍 MrX-Tool


MrX-Tool est un outil simple avec des fonctions OSINT basiques et des raccourcis vers des ressources Discord.

⚙️ Fonctionnalités
🔎 1. Recherche de pseudo

Permet de rechercher un pseudo sur différentes plateformes pour voir sa présence en ligne.

🌐 2. Server Lookup (Discord Informatique)

Le module Server Lookup sert uniquement à accéder rapidement à un serveur Discord lié à l’informatique.

👉 Fonction :

Redirige ou affiche un lien vers un serveur Discord
Permet de rejoindre un serveur informatique rapidement
